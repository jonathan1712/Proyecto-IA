************************************* PRUEBAS CON PYTEST-COV ******************************************************

*Para correr las pruebas de pytest-cov primero se debe instalar con el comando:

    $pip install pytest-cov

*Para correr coverage en un archivo solo basta escribir en consola:

    $coverage run filename.py

*Si se desea crear un html a partir de el último análisis realizado basta con:

    $converage html

*Con pytest es similar:

    $pytest --cov filename.py
