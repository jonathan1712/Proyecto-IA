##Carpeta HTMLs

*En esta carpeta se encuentran los resultados de pytest coverage en html a partir de los archivos de test_file*.py

*Para observar los resultado obtenidos basta con ingresar a la carpeta respectiva y abrir con el navegador el archivo index.html

*En este archivo se pueden observar los distintos datos que nos ofrese este análisis (statements, missing, excluded, coverage) en una tabla, si se desea indagar más a fondo y ver aquellas líneas que fueron o no exesadas por el test se puede dar clic sobre los distintos módulos (.py) que la tabla presenta

*En los módulos aparte se puden observar las líneas de código corridas, perdidas y excluidas.


**Nota**
Los archivos .coverage y .coveragerc pertenecen a esta librería
